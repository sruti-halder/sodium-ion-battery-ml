# Sodium-Ion Battery ML Project

## Overview

This project applies machine learning techniques to predict key electrochemical properties of sodium-ion battery electrode materials, including voltage, capacity, and volume change.

## Workflow

Data → Cleaning → Feature Engineering → Model Training → Validation → Screening

## Project Structure

* data/ : contains raw and processed datasets
* src/ : contains all modular Python scripts
* main.py : executes the complete pipeline

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## Models Used

* Random Forest
* Gradient Boosting
* XGBoost

## Output

* Prediction of voltage, capacity, and volume change
* External validation using literature datasets
* Screening of new candidate materials
